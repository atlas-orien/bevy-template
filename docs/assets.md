# 资源记录

`docs/assets.md` 只记录项目真实引入的第三方资源来源和许可证。

资源目录规则写在：

- `assets/README.md`
- `AI_PROTOCOL/ASSETS.md`

## 记录格式

引入第三方资源时，按项目需要添加记录：

- runtime 路径
- 来源 URL 或本地来源路径
- 许可证
- 用途
